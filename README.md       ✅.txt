Este proyecto corresponde a la Parte I del Proyecto Programado del curso Programación I.  
El objetivo es crear las clases necesarias para modelar el proceso de elecciones presidenciales en Costa Rica, aplicando conceptos de programación orientada a objetos en Java.

Clases implementadas
- PartidoPolitico
- Candidato
- Votante

Cada clase cuenta con:
- Atributos privados
- Constructor
- Métodos getters y setters
- Método `toString()`

Programa principal
El programa principal muestra un menú con las siguientes opciones:
1. Incluir un partido  
2. Incluir un candidato  
3. Incluir un votante  
4. Registrar un voto  
5. Salir  

El menú se repite hasta que el usuario elige salir.  
Por indicación del enunciado, las opciones muestran el mensaje “En construcción”.

Observación
Este proyecto corresponde únicamente a la primera entrega.  
Las funcionalidades completas se desarrollarán después, a lo que intuimos.
